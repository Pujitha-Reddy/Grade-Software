# Grade-Software
Implement software to generate a Letter Grade for a course taken by a student.

ABOUT THE PROJECT

The goal of this Project is to explore the issues related to Analysis, Design, Implementation & Testing a
final Product to locate possible BUGS leading to software failure.

Facts

1. Assume you are to design and implement software to generate a Letter Grade for a course taken
by a student.
2. The course syllabus indicates the following grade break down:
   a. Homework 15% of the Total Grade
   
   b. Quiz 5 % of the Total Grade
   
   c. Mid-Term Exam 25% of the Total Grade
   
   d. Final Exam 30% of the Total Grade
   
   e. Final Project 25% of the Total Grade
   
4. Assume that there were 8 homework assignments, 5 quizzes, one Mid-Term Exam, One Final
Exam, and One Final Project.

Task:
   1. Give a scenario of a student data.
   2. Evaluate manually the final grade obtained by the student.
   3. Implement and test your Code.
  
Requirements:

   1. Identify the Problem solved by your Product.
   2. Your Final Product should exhibit the following features:
      
      a. Functionality

      b. Usability

      c. Reliability

      d. Performance

      e. Extentandability
